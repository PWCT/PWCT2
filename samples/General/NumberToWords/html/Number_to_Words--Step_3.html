Step 3




3. Digits from 1 to 999 (Hundreds)




Now we will convert numbers 1..99 to words.
In order to evaluate complete groups of 3 digits, we need a small extension - the Hundreds.
How does this work? Simple. The first number of a triplet is always the number of Hundreds.
...
elseif len (n) = 3 // first place: hundreds

The "Hundreds" are needed in each group.
There are not only a hundred as such, but also a hundred thousand or a hundred thousand million, but also a hundred thousand billion.
Our program should be able to convert as long as you want.
For this we need a Recursive (self-calling) function.

Here it is the function: doWord ()

// the zeros in the groups or whole null groups are not treated.
