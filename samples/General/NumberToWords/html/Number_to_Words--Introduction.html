Analysis

We will tackle the task in each step and check them continuously

:

Step 1

- Digits from 1 to 19



Topics: Array, input, input control (if, then, else), output.
We start off casually and write a piece of code.
The task is to convert a number between 1..19 into its words.
Inspired by the success, we continue.


Step 2

- Digits from 20 to 90



Topics: The same as in Step 1
Extension of the code from Step-1 using whole numbers - the Tens groups: 20..90.


Step 3

- Digits from 1 to 999 (Hundreds)



Topics: The same as before plus Functions with Recursion techniques.
Let's deal with the representation of Hundreds (the first of a group of three).
And the solution using a recursive function (it calls itself, repeat).


Step 4

- Thousands / Millions Groups



Topics: Same as before plus division of the input into groups.
Break down the long number into individual groups of three digits each.
Dividing them into Thousands, Millions and so on.


Step 5

- Output of the whole Number as Word



Topics: Assembly and testing
Assemble all the previous program steps created so far.


Step 6

- Program with graphical Interface



Topic: The same as Step 5, with the addition of a Graphical UI and small extensions.
