Preamble




Programming is no different than splitting a problem into several smaller ones, and then solving each smaller piece.
As an example let's assume we want to represent any number in words - one hundred twenty three for 123.

Writing a small program that gives the results you want is not that difficult.
One that can be created properly and easily, to expand it takes several aspects into account.

Suppose we want to write an application that converts a number to words.
For example the number: 1,234,567 as:

one million two hunderd thousand five hunderd sixty seven

Let's address the problem with the following steps:

1. Think about the problem.
3. Break up the problem into smaller parts and write it down formally.
4. Convert the parts into the computer language you are using (pseudo code?).
5. Test the individual parts for functionality.
6. Put the parts together.

How do we start?

Step by step, One step after another..
