==================
Console Components
==================

In this chapter we are going to learn about the Console Components

.. index:: 
	pair: Console Components; Introduction

Introduction
============

These components could be used for Input/Output operations in console applications

.. image:: images/consolecomponents/cb_consolecomponents.png
        :alt: console components

We have four components in this category:

* Print Text
* Get Input
* Get Character
* Input() Function  

.. index:: 
	pair: Console Components; Print Text

Print Text
==========

Using this component we can print text on the screen

Also we can print the result of an expression

.. image:: images/consolecomponents/ib_console_printtext.png
        :alt: Print Text

The Interaction Page contains the next properties

============	==============	==============
Property	Type		Default Value
============	==============	==============
Text		Textbox		none
Type		Listbox		Literal
New Line	Checkbox	True
============	==============	==============

.. index:: 
	pair: Console Components; Get Input

Get Input
=========

Using this component we can get input from the User using the Keyboard

The entered text will be stored in memory

We can access this text later using the Variable Name

.. image:: images/consolecomponents/ib_console_getinput.png
        :alt: Get Input

The Interaction Page contains the next properties

=================	==============	==============
Property		Type		Default Value
=================	==============	==============
Variable		Textbox		cInput
Convert to Number	Checkbox	False
=================	==============	==============

.. index:: 
	pair: Console Components; Get Character

Get Character
=============

Using this component we can get one character from the User using the Keyboard

The entered character will be stored in memory

We can access this character later using the Variable Name

.. image:: images/consolecomponents/ib_console_getcharacter.png
        :alt: Get Character

The Interaction Page contains the next properties

=================	==============	==============
Property		Type		Default Value
=================	==============	==============
Variable		Textbox		c
=================	==============	==============


.. index:: 
	pair: Console Components; Input() Function

Input() Function
================

Using this component we can get some of characters from the User using the Keyboard

The entered characters will be stored in memory

We can access these characters later using the Variable Name

.. image:: images/consolecomponents/ib_console_inputfunction.png
        :alt: Input function

The Interaction Page contains the next properties

=================	==============	==============
Property		Type		Default Value
=================	==============	==============
Characters Count	Textbox		30
Output (Variable)	Textbox		cValue
=================	==============	==============
